# Aahaas Ecommerce

A modern ecommerce platform built with React and Laravel.

## Features

- Product catalog
- Shopping cart
- Responsive design
- Laravel API integration

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Make sure Laravel API is running on port 8000

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Laravel Backend
